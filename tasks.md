# Danh sách Task: Tích hợp game Cờ Ca rô (Gomoku) vào Web

Dưới đây là danh sách chi tiết các công việc cần thực hiện để thêm game Cờ Ca rô vào web, bao gồm thiết kế giao diện, logic game, thuật toán AI và kiểm thử.

## Danh sách Task Items

- [ ] **Task 1**: Thiết lập cấu trúc giao diện trong `index.html` (Bổ sung tab chuyển đổi game, giao diện chứa game mới, nút điều khiển và load caro.js).
- [ ] **Task 2**: Cập nhật CSS trong `style.css` (Style cho tab chuyển đổi game, grid bàn cờ 12x12, phong cách neon cho quân cờ X/O, hover helper và animation).
- [ ] **Task 3**: Cập nhật logic chuyển game trong `main.js` (Điều hướng ẩn/hiện game container, dừng game đang chơi và dừng bộ đếm thời gian khi chuyển đổi tab).
- [ ] **Task 4**: Phát triển Core Engine Caro trong `caro.js` (Bàn cờ 12x12, quản lý lượt chơi PvP, thuật toán kiểm tra 5 quân thắng cuộc, hiển thị highlight hàng thắng cuộc và bảng điểm).
- [ ] **Task 5**: Triển khai thuật toán AI cho chế độ PvE trong `caro.js` (Thuật toán đánh giá Heuristic, quét vùng lân cận, chặn/công cơ bản và giả lập độ trễ nước đi 300-500ms).
- [ ] **Task 6**: Tinh chỉnh UI/UX và Đánh giá QA (Tối ưu giao diện mobile responsive, kiểm thử các điều kiện biên và hoàn thành QA/Review).
